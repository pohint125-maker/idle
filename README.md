# Idle Moto Empire

Mini gioco idle in italiano sul mondo delle moto.

## Feature
- Clicca **Fai un giro** per ottenere crediti.
- Compra **Moto** per produzione automatica.
- Assumi **Crew** per aumentare il guadagno per click.
- Apri **Officine** per moltiplicare la produzione delle moto.
- Espandi **Garage** per bonus globali.
- Salvataggio automatico su `localStorage`.

## Avvio locale
```bash
python -m http.server 4173
```
Poi apri `http://localhost:4173`.
